## ðŸ“Œ Installation

**1. Clone the repository**

```bash
git clone git@github.com:mostafizurhimself/admintoolkit-html.git
```

**2. Install dependencies**

```bash
yarn install
```

**3. Start development server**

```bash
yarn dev
```

**4. Build for production**

```bash
yarn build
```

**5. Preview production build**

```bash
yarn start
```
